# NextLevel Food

Simple full-stack web app built with NextJS and SQLite, it allows you to browse food recipes or add your own one !

## Tech Stack

This app is built with NextJS and SQLite.

## Setup

```bash
npm install
npm run dev
```

## Features

- Browse food recipes
- Add your own recipe
